API Reference

Integrate AFKmate analysis into your workflow.

Overview

The AFKmate API allows you to programmatically analyze code and retrieve results. This is useful for CI/CD pipelines, custom integrations, or building your own tools on top of AFKmate.

Base URL: https://api.afkmate.dev/v1

Authentication

All API requests require an API key. Include it in the header:

Authorization: Bearer YOUR_API_KEY

Get your API key from the Premium dashboard.

Endpoints

POST /analyze

Analyze a code snippet or file.

{
  "code": "function example() { ... }",
  "language": "javascript",
  "filename": "example.js"
}

GET /usage

Get your current usage statistics.

GET /v1/usage
Authorization: Bearer YOUR_API_KEY

GET /health

Check API health status (no authentication required).

Rate Limits

  • Free tier: 10 requests per 15 days
  • Premium: Unlimited requests
  • Team: Unlimited requests + priority queue

SDKs

Official SDKs coming soon for:

  • Node.js / TypeScript
  • Python
  • Go

In the meantime, you can use any HTTP client to interact with the API.
