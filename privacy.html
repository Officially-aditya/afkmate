Privacy Policy

How we collect, use, and protect your information.

Last updated: January 2025

Introduction

AFKmate ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard your information when you use our VS Code extension and related services.

Information We Collect

Code Analysis Data

When you use AFKmate's cloud analysis feature, we temporarily process your code to provide analysis results. This code is:

  • Transmitted securely via HTTPS
  • Processed in memory only
  • Not stored on our servers after analysis
  • Never shared with third parties

Usage Data

We collect anonymous usage statistics including:

  • Number of analyses performed
  • Feature usage patterns
  • Error reports (without code content)

Account Information

For premium users, we collect:

  • Email address
  • Payment information (processed by Stripe)

Local Analysis Option

AFKmate supports local analysis via Ollama. When using local mode:

  • Your code never leaves your machine
  • No data is sent to our servers
  • Analysis is performed entirely locally

How We Use Your Information

We use collected information to:

  • Provide and improve our analysis services
  • Process payments and manage subscriptions
  • Send important service updates
  • Respond to support requests

Data Security

We implement industry-standard security measures:

  • TLS 1.3 encryption for all data in transit
  • No persistent storage of analyzed code
  • Regular security audits
  • Secure payment processing via Stripe

Your Rights

You have the right to:

  • Access your personal data
  • Request data deletion
  • Opt out of analytics
  • Use local-only analysis mode

Contact Us

For privacy-related questions, contact us at privacy@afkmate.dev
