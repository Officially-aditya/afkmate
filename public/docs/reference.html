Reference

Commands, troubleshooting, and where to get help.

Commands

All commands are available from the VS Code command palette (Cmd/Ctrl+Shift+P):

  • AFKmate: Analyze Now Run analysis immediately
  • AFKmate: Pause Analysis Temporarily disable idle detection
  • AFKmate: Resume Analysis Re-enable idle detection
  • AFKmate: Check Usage View remaining free analyses
  • AFKmate: Export Results Save results to file
  • AFKmate: Copy Results Copy results to clipboard
  • AFKmate: Fix All Issues Generate fixes for all findings
  • AFKmate: Undo Last Fix Revert the last applied fix

Troubleshooting

Analysis not starting

  • Check the status bar for a paused indicator
  • Verify you have remaining free analyses (or use Ollama for unlimited)
  • Ensure internet connection if using cloud provider

Slow analysis

  • Large files take longer to process
  • Check your internet connection speed
  • Try local Ollama for faster results on smaller models

No results showing

  • Check if error/warning/suggestion filters are enabled in settings
  • Verify the file type is supported
  • Try running AFKmate: Analyze Now manually
